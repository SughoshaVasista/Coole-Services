# Cool-E: Online Urban Service Marketplace

Cool-E is a sophisticated service marketplace platform built with Django and MySQL. It connects customers with professional service providers through a seamless, secure, and aesthetically pleasing interface.

## 📄 Documentation for Judges
We have prepared a detailed documentation for the presentation.
Please refer to: **[JUDGES_DOCUMENTATION.md](./JUDGES_DOCUMENTATION.md)**

## 🌟 Key Highlights
- **Real-time Availability**: Anti-double-booking logic.
- **UPI Integration**: Native QR-based payment flow.
- **Partner Dashboard**: Comprehensive job management for service providers.
- **Premium Design**: Modern UI with glassmorphism and smooth animations.

## 🚀 Quick Start
```bash
# Install dependencies
pip install -r requirements.txt

# Run migrations
python manage.py migrate

# Start development server
python manage.py runserver
```
